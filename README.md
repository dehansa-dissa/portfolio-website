# Dehansa Dissanayake — Portfolio
 
A personal portfolio website built with **React + Vite + TypeScript + Tailwind CSS**. Single-page design with smooth scroll navigation, dark/light theme, and reveal-on-scroll animations.
 
---
 
## Tech Stack
 
- **React 18** — UI library
- **Vite 6** — build tool & dev server
- **TypeScript** — type safety
- **Tailwind CSS v3** — utility-first styling
- **Lucide React** — icons
- **clsx + tailwind-merge** — conditional class utilities

---

## Features
 
- ✅ Dark / light theme toggle
- ✅ Typewriter animation cycling through roles
- ✅ Scroll-based active nav link highlighting
- ✅ Reveal-on-scroll animations (IntersectionObserver)
- ✅ Floating dots & grid background in Hero
- ✅ Card hover effects with glow
- ✅ Responsive — mobile menu included
- ✅ Contact form with sent confirmation state
- ✅ Back-to-top button
